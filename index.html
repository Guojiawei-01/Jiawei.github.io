Featured Project

Does Financial News Sentiment Move the S&P 500?

Repository Final report

This DATA 498D group capstone tested whether daily financial news headline sentiment contains useful information about next-day S&P 500 movement. The project compared dictionary sentiment, VADER, FinBERT, topic features, Logistic Regression, XGBoost, and long/flat backtests.

The final finding was deliberately cautious: daily headline sentiment showed weak predictive power, and backtest performance depended heavily on regime, threshold choice, and trading assumptions. That result made the project stronger because it separated a clean workflow from an overstated trading claim.

My role: data preparation lead

  • Validated the raw Kaggle headline file and documented data limitations.
  • Removed exact duplicates and built the daily aggregation pipeline.
  • Constructed next-day return and direction targets for modeling.
  • Integrated FRED macro context including VIX, Treasury rates, CPI, unemployment, and recession indicators.
  • Maintained the data dictionary and EDA summaries used by the team.

Time-aware validation

Train, validation, test, and expanding-window splits were designed to reduce leakage in market prediction.

Method comparison

Dictionary, VADER, FinBERT, and combined feature sets were compared instead of relying on one sentiment score.
